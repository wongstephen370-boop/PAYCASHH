# PayCash

A payment app for Jamaican entrepreneurs.

## Setup

1. Clone the repo.
2. For backend: cd backend, npm install, npm start.
3. For mobile: cd mobile-app, npm install, expo start.
4. For admin: Open admin-dashboard/index.html in browser.

## Deployment

- Backend: Use DigitalOcean App Platform, set source to backend.
- Mobile: Build with Expo.
- Admin: GitHub Pages.
